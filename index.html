Introduction to C

The C programming language was developed in the early 1970s by Dennis Ritchie at Bell Labs. It was originally designed for writing the Unix operating system, but its efficiency and portability made it one of the most influential programming languages in history.

C is a procedural, compiled language that provides low-level access to memory and hardware, making it ideal for system-level programming such as operating systems, embedded systems, and device drivers.

#include int main() { printf("Hello, World!\n"); return 0; }
Data Types and Variables

C supports several built-in data types including int, float, double, char, and _Bool. These types define the kind of data a variable can hold and how much memory it occupies.

Variables must be declared before use, specifying their type and name. C uses static typing, meaning the type is checked at compile time and cannot change.

int age = 25; float price = 19.99; char grade = 'A';
  • int: Used for integers.
  • float: Single-precision floating-point number.
  • char: Stores a single character.
Control Structures

C provides standard control-flow constructs such as conditionals and loops. These allow programs to make decisions and repeat actions.

The if, else if, and else statements are used for branching logic. Loops like for, while, and do-while enable repetition.

if (score >= 90) { printf("Grade: A\n"); } else { printf("Grade: B or lower\n"); }
  • for loops are ideal when the number of iterations is known.
  • while loops continue while a condition is true.
Functions

Functions in C are blocks of code that perform a specific task. Every C program must have a main() function, which serves as the entry point.

Functions improve code modularity and reusability. They can accept parameters and return a value (or void if none).

int add(int a, int b) { return a + b; }

To use a function, it must be declared (or defined) before it is called. Function prototypes are often placed at the top of the file.

Memory Management

Unlike languages with garbage collection, C requires manual memory management using functions from .

Dynamic memory is allocated on the heap using malloc(), calloc(), or realloc(), and must be explicitly freed using free() to avoid memory leaks.

int *arr = malloc(5 * sizeof(int)); if (arr != NULL) { arr[0] = 1; free(arr); // Important! }

Proper memory management is crucial in C programming to ensure stability and efficiency.
